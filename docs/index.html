Features

Everything you need. Nothing you don't.

Live window thumbnails
Real-time captures via ScreenCaptureKit — every thumbnail shows the window's actual current state, not a stale screenshot.
Spotify panel
Album art, track info, progress bar and full playback controls — all surfaced at the Dock, without switching to the app.
Not running? One tap.
For dormant Dock icons, DockDock shows a placeholder with a direct Open button. Launch any app without leaving the Dock.
Zero footprint, native only
Lives in the menu bar, no Dock icon. Built entirely with public Apple APIs — Accessibility, ScreenCaptureKit, AppKit. No private entitlements.
Install

Two commands.
You're done.

Clone the repo and run the build script. It compiles a signed release binary and installs it to /Applications — under a minute from zero.

1
On first launch, grant Accessibility when macOS prompts. Required to read Dock icon positions and manage windows.
2
Grant Screen Recording in System Settings → Privacy & Security for live window thumbnails.
3
DockDock sits in the menu bar — no Dock icon, no background windows. The preview panel appears only when you hover.

Requires Xcode Command Line Tools (xcode-select --install) or Xcode. After each new build, macOS may ask to re-grant Screen Recording — the permission is tied to the binary hash.
